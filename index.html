< About Me >

I am Md. Nahidul Islam, also known as fumioryoto, a cybersecurity enthusiast and developer with a strong interest in web application security and penetration testing. I enjoy exploring how modern applications work behind the scenes and analyzing how security weaknesses appear within real systems. My work involves studying web vulnerabilities, attack techniques, and defensive security practices, while experimenting with different tools and environments used in security research. I also develop small utilities and scripts to support automation, testing, and learning in offensive security. Beyond security testing, I regularly work with programming and development technologies, which helps me better understand application architecture and how vulnerabilities originate during development. This development background allows me to approach security assessments with both an attacker’s mindset and a developer’s perspective. I am particularly interested in ethical hacking, vulnerability research, and security operations, and I continuously expand my knowledge by experimenting with new technologies, frameworks, and attack surfaces to understand how systems can be designed and secured against real-world threats.
